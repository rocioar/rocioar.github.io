Optimizing Django ORM Queries

2020-05-03 :: Rocio Aramberri
The Django ORM (Object Relational Mapping) is one of the most powerful features of Django, it enables us to interact with the database using Python code. It has multiple advantages: The database engine is abstracted from us so it is possible to switch to another database system with ease. Supports migrations: we can easily change our tables by updating our models and Django will automatically generate the migration scripts needed to update our database tables.
